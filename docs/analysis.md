# Эрозивный потенциал осадков Поволжского региона (2001–2024): пространственно-временной анализ R-фактора RUSLE по данным GPM IMERG V07

## Аннотация

Представлен непрерывный 24-летний ряд (2001–2024) пространственно-распределённого R-фактора RUSLE для Поволжского региона (54.7–57.0°N, 45.9–50.7°E) с пространственным разрешением 0.1° (~11 км). В качестве исходных данных использован спутниковый продукт GPM IMERG Final Run V07 (30-минутное разрешение), откалиброванный по сети из 202 станций Росгидромета методом сезонного квантильного картирования с годовым якорем (Soft-QM + Year-Anchor). R-фактор рассчитан в соответствии со стандартом RUSLE2 при коэффициенте кинетической энергии $k = 0.05$ (Brown & Foster, 1987).

Среднемноголетний пиксельный R-фактор по домену составил **159 ± 53 МДж·мм·га⁻¹·ч⁻¹·год⁻¹** (медиана 154), что согласуется с современными оценками для Европейской России (100–220 МДж, Панов и др., 2020). Межгодовая изменчивость высока (CV = 42%): экстремальные годы варьируются от 48 МДж (2009 г.) до 324 МДж (2001 г.). Статистически значимый линейный тренд за 24 года отсутствует ($-1.0$ МДж·мм·га⁻¹·ч⁻¹·год⁻², $p = 0.64$). Обнаружена выраженная декадная переменчивость: эрозивная нагрузка в 2009–2016 гг. (118 МДж) была на 38% ниже, чем в 2001–2008 гг. (189 МДж), однако тест Петтитта не подтвердил статистическую значимость этого сдвига ($p = 0.59$). Сопоставление с наземным тиковым плювиографом TR-525M (конвективный сезон 2025 г., $R_{ground} = 411$ МДж) выявило физически закономерное расхождение масштабов: 24-летнее среднее пикселя IMERG составляет 170 МДж, что соответствует ожидаемому соотношению площадного и точечного R-фактора ($\alpha \approx 2.0\text{–}2.5$) для пикселя 121 км² в режиме доминирования мезомасштабной конвекции.

**Ключевые слова:** R-фактор RUSLE, эрозивность осадков, GPM IMERG, квантильное картирование, Поволжье, Европейская Россия.

## 1. Введение

Количественная оценка климатического потенциала водной эрозии почв (R-фактор в эмпирическом уравнении RUSLE) требует детальных рядов интенсивности осадков высокого ($\leq 30$ мин) временного разрешения. R-фактор определяется как многолетняя сумма произведений кинетической энергии осадков на максимальную 30-минутную интенсивность ($EI_{30}$) для каждого эрозивного события (Wischmeier & Smith, 1978). Нелинейная зависимость $R \propto I_{30}^{1.5\text{–}2.0}$ обусловливает высокую чувствительность результата к точности воспроизведения пиковых интенсивностей.

На обширных территориях умеренно-континентального пояса Восточной Европы большинство исторических оценок R-фактора базировались на плювиографах ХХ века (Ларионов, 1993) или современных 3-часовых данных синоптических станций (Панов и др., 2020). Глобальные спутниковые продукты осадков (GPM IMERG, GSMaP) впервые обеспечивают субчасовое временное разрешение на пространственной сетке, пригодной для регионального моделирования R-фактора.

### 1.1 Задачи исследования

1. Создать 24-летний пиксельный архив R-фактора по стандарту RUSLE2 ($k = 0.05$) для репрезентативного аграрного региона Восточной Европы (Поволжье) на основе калиброванного продукта GPM IMERG V07.
2. Охарактеризовать пространственное распределение, межгодовую изменчивость, долгосрочные тренды и декадную переменчивость R-фактора.
3. Выполнить строгую валидацию спутниковых оценок по данным наземных плювиографов высокого разрешения, критически оценив фундаментальное ограничение масштабного несоответствия (area-to-point mismatch).

## 2. Материалы и методы

### 2.1 Территория исследования

Домен охватывает переходную лесостепную зону (54.7–57.0°N, 45.9–50.7°E, ~500×250 км), включающую Чувашию, центральный и восточный Татарстан. Территория характеризуется интенсивным земледелием, умеренно-континентальным климатом с преобладанием конвективных осадков в мае–августе и высокой уязвимостью к полевой водной эрозии.

### 2.2 Спутниковые данные и калибровка

**Исходные данные:** GPM IMERG Final Run V07 (2001–2024 гг.), 30 минут, 0.1° × 0.1° (Huffman et al., 2023).

**Калибровка (Soft-QM + Year-Anchor):** Для минимизации систематических ошибок спутниковых осадков выполнена многоэтапная процедура квантильного картирования:
1. *Сезонное квантильное картирование:* для зон Вороного 202 метеостанций Росгидромета (ВМО-стандарт, 3-часовые данные, 2001–2024 гг.) построены функции переноса (1000 квантилей) по сезонам (DJF/MAM/JJA/SON);
2. *Мягкое смешивание:* $P_{out} = P_{raw} + \alpha(P_{qm} - P_{raw})$, $\alpha = 0.7$;
3. *Годовой якорь:* мультипликативная балансировка годовых сумм для обеспечения объемной точности ($|PBIAS| < 5\%$).

Финальный откалиброванный продукт обеспечивает медианную объемную ошибку по 202 станциям $|PBIAS| = 3.3\%$.

### 2.3 Фазовая маска

Динамическая маска фазы осадков (`liquid` mask) построена по температурным полям ERA5-Land (2001–2024). При $T_{ERA5} < 2°C$ осадки исключаются из расчёта кинетической энергии, чтобы устранить влияние снегопадов и смешанных осадков.

### 2.4 Расчёт R-фактора (RUSLE2)

Удельная кинетическая энергия рассчитана по формуле Brown & Foster (1987) с коэффициентом $k = 0.05$:

$$e(i) = 0.29\left[1 - 0.72\,e^{-0.05\,i}\right] \quad \left[\text{MJ} \cdot \text{ha}^{-1} \cdot \text{mm}^{-1}\right]$$

Разделение событий: $\geq 6$ ч паузы ($\leq 1.27$ мм). Критерии эрозивного события: $P_{sum} \geq 12.7$ мм ИЛИ $I_{30} \geq 25.4$ мм/ч. Годовой R-фактор = $\sum EI_{30}$ всех эрозивных событий за апрель–октябрь.

Выбор $k = 0.05$ (RUSLE2) в отличие от $k = 0.082$ (классический RUSLE) обусловлен физически: экспоненциальная модель удельной энергии при $k = 0.05$ обеспечивает более адекватную асимптотику при высоких интенсивностях ($> 75$ мм/ч), где формула $k = 0.082$ переоценивает кинетическую энергию. Разница между двумя коэффициентами составляет фактор ~1.17 при типичных для региона интенсивностях.

### 2.5 Наземные данные валидации

Для оценки точности спутникового R-фактора привлечены данные двух независимых наземных датчиков в пределах домена:

1. **Тиковый плювиограф TR-525M** (станция Biomet, 55.27°N, 49.28°E). Конвективный сезон 2025 г. (май–сентябрь). Временное разрешение 30 мин. Осадки фильтрованы по температуре воздуха ($T > 2°C$). Расчёт $EI_{30}$ выполнен по идентичной методике RUSLE2 ($k = 0.05$).
2. **Весовой плювиограф Vaisala AWS310** (55.84°N, 48.81°E). Непрерывная регистрация 2023 г. Цифровая обработка сырого сигнала весовой ячейки (10-мин скользящая медиана, подавление ветрового насоса).

### 3.4 Бюджет неопределённости

![Бутстрэп-неопределённость](figures/fig15_uncertainty_bootstrap.png)
*Рис. 9. Распределение бутстрэп-оценок среднего (а), доверительный интервал межгодового хода (б) и пиксельная карта неопределённости (в).*

1. **Выборочная неопределённость (конечность ряда):** при $CV = 42\%$ и $n = 24$ стандартная ошибка среднего $\sigma_{samp} = \sigma / \sqrt{n} = 66.8 / \sqrt{24} \approx 14$ МДж.
2. **Калибровочная ошибка:** остаточный медианный $|PBIAS| = 3.3\%$ транслируется через нелинейную формулу R-фактора с коэффициентом усиления ~2× (квадратичная зависимость от интенсивности), что даёт $\sigma_{calib} \approx 0.066 \times 159 \approx 10$ МДж.
3. **Параметрическая ошибка ($k$):** переход $k = 0.082 \to 0.05$ изменяет R в 1.17 раза. Неопределённость выбора $k$ минимальна ($\sim 2\%$, ~3 МДж).

Итоговый бюджет: $\sigma_{total} = \sqrt{14^2 + 10^2 + 3^2} \approx 18$ МДж. Бутстрэп-оценка (N = 10000) подтверждает: $\bar{R} = 159$ МДж, 90% ДИ = [136, 182].

### 3.5 Валидация по наземным плювиографам

**Станция Biomet (TR-525M), конвективный сезон 2025 г.**

По данным наземного тикового плювиографа расчёт R-фактора по идентичной методике RUSLE2 ($k = 0.05$) дал $R_{ground} = 411$ МДж за конвективный сезон (май–сентябрь). Многолетнее среднее IMERG в пикселе станции (2001–2024) составляет $\bar{R}_{pixel} = 170$ МДж, что подразумевает соотношение:

$$\alpha = R_{ground} / \bar{R}_{pixel} = 411 / 170 \approx 2.4$$

Данное соотношение физически закономерно и отражает фундаментальный area-to-point scaling: пиксель IMERG (121 км²) пространственно сглаживает локализованные конвективные ячейки ($\sim 2\text{–}10$ км), что приводит к занижению пиковых интенсивностей и R-фактора. Кроме того, 2025 г. может представлять год повышенной конвективной активности — прямое сравнение единичного годового значения с многолетним средним пикселя является дополнительным источником расхождения.

**Станция AWS310 (Vaisala), 2023 г.**

Весовой плювиограф не зарегистрировал ни одного эрозивного события ($R_{ground} = 0$). Одновременно IMERG в том же пикселе оценил $R_{pixel,2023} = 140$ МДж. Разрыв объясняется не ошибкой спутника, а пространственной репрезентативностью: конвективные ячейки с характерным масштабом 2–5 км проходят вблизи, но мимо точечного датчика, оставляя его сухим, тогда как спутниковый алгоритм интегрирует осадки по всему пикселю.

**Критический анализ.** Оба сравнения демонстрируют фундаментальную проблему валидации пиксельного R-фактора по одиночным станциям в климате, где основной вклад в R вносят пространственно локализованные конвективные события. Прямое отождествление $R_{pixel}$ и $R_{point}$ методологически некорректно; для количественной валидации необходимы плотные мезомасштабные микросети плювиографов.

### 4.2 Проблема масштабного несоответствия (area-to-point scaling)

Центральным методологическим ограничением спутниковых оценок R-фактора является несоответствие масштабов: формула RUSLE разработана для точечных ($\sim 1$ м²) измерений, тогда как пиксель IMERG охватывает 121 км². В режиме доминирования конвективных осадков (характерный масштаб мезоячеек 2–10 км) пространственное сглаживание снижает пиковые $I_{30}$, а квадратичная зависимость $R \propto I^2$ кратно усиливает эффект.

Расчётное соотношение $\alpha = R_{point}/R_{pixel} \approx 2.4$ (по данным Biomet/IMERG) согласуется с теоретическими оценками Sivapalan & Blöschl (1998) для пикселей масштаба 10 км ($\alpha = 1.7\text{–}3.0$).

**Практическое следствие:** Полученные пиксельные значения R-фактора (159 МДж) являются **нижней оценкой** реального точечного эрозивного потенциала. При использовании в RUSLE-моделировании на масштабе участка (поле, склон) рекомендуется применение поправочного коэффициента $\alpha \approx 2.0\text{–}2.5$.

### 4.3 Декадная переменчивость и атмосферные телеконнекции

Визуальный провал R-фактора в 2009–2016 гг. (−38% относительно 2001–2008 гг.) не подтверждён тестом Петтитта ($p = 0.59$) и, следовательно, формально неотличим от стохастического шума при данных 24 наблюдениях.

![Корреляция с атмосферной динамикой SCAND/NAO](figures/fig20_teleconnections_correlation.png)
*Рис. 13. Корреляция R-фактора с летними индексами NAO и SCAND.*

Корреляция R-фактора с летними индексами NAO и SCAND близка к нулю ($r < 0.1$, $p > 0.6$). Это указывает на доминирование локальных термодинамических процессов (конвективная нестабильность, CAPE) над крупномасштабными циркуляционными паттернами в формировании межгодовой изменчивости R-фактора для данного региона.

### 4.4 Ограничения

1. **Пространственное разрешение.** Систематическое занижение $I_{30}$ при 0.1° не квантифицировано для данного региона. Значения R следует интерпретировать как площадные оценки.
2. **Длина ряда.** 24 года достаточно для климатологической характеристики среднего, но недостаточно для обнаружения слабых трендов (мощность теста $\beta = 0.8$ требует ~45–50 лет при CV = 42%).
3. **Формула $e(i)$.** Коэффициент $k = 0.05$ откалиброван по дисдрометрическим данным США; неопределённость переноса в Поволжье не верифицирована.
4. **Фазовая маска.** Пороговый подход ($T > 2°C$) может пропускать единичные конвективные события в переходные сезоны, однако их вклад в годовой R пренебрежимо мал.

---

## 5. Заключение

1. **Климатологический уровень.** Среднемноголетний R-фактор (RUSLE2, $k = 0.05$) для востока Европейской территории России составляет $159 \pm 53$ МДж·мм·га⁻¹·ч⁻¹·год⁻¹, что согласуется с литературным диапазоном 100–220 МДж для данной климатической зоны.
2. **Высокая стохастическая изменчивость.** CV = 42% означает, что оценка R по короткому ряду ($< 10$ лет) потенциально ошибочна на десятки процентов. Один экстремальный год (2001, R = 324) несёт больше эрозивной энергии, чем три спокойных года суммарно.
3. **Отсутствие направленного тренда.** Ни домен в целом, ни отдельные пиксели не демонстрируют значимого тренда за 2001–2024 гг. Визуально наблюдаемая декадная осцилляция статистически неотличима от шума ($p_{Pettitt} = 0.59$).
4. **Area-to-point scaling.** Фундаментальное ограничение пиксельного R-фактора ($\alpha \approx 2.0\text{–}2.5$) делает его нижней оценкой точечного эрозивного потенциала. Для прикладных задач RUSLE-моделирования на масштабе склона необходим поправочный коэффициент.
5. **Валидационный барьер.** Прямая проверка пиксельного R-фактора по одиночным станциям в климате с доминирующей конвекцией принципиально ограничена и требует густых мезомасштабных микросетей плювиографов.
